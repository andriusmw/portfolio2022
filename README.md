# portfolio2022
repo actualizado de mi portfolio de este año como front-end web developer. Es Responsive y Destacaría la función onclick para mostrar y ocultar las partes del portfolio sin tener que recargar la página.


## Comenzando 🚀

Para levantar el proyecto en modo local solo hay que descargar el repositorio y abrirlo con tu IDE preferido, levantar un servidor en localhost:4000 
y listo.


### Pre-requisitos 📋

Se recomienda usar Visual Studio Code para importar el proyecto y levantar el servidor.

### Instalación 🔧

una vez descargado y abierto el proyecto con Visual Studio Code; lanzamos en una terminal los siguientes comandos.

cd <nombre de la carpeta del proyecto>
ng serve --open
  
Con esto debería de cargarnos una ventana en el navegador a la ruta localhost/:4000 
para que funcioenen correctamente las funciones de javascript.
